# Student Management System

This project is a console-based Student Management System developed using Core Java and MySQL.

## Features
- Add student records
- View student records
- Delete student records
- Auto-generated student ID
- Secure SQL queries using PreparedStatement

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Description
The system allows users to manage student information through a menu-driven console interface.
Student data is stored in a MySQL database and accessed using JDBC connectivity.
